## 她是谁？

达妮娅是《鸣潮》中登场的五星角色——星炬学院虚质科学部学生，曾被残星会改造为鸣式「阿列夫一」的共鸣容器。在游戏剧情中，她最终选择自我放逐，困于虚质空间。

这个项目试图续写她的故事：在虚质空间中，她意外连接到了我们的世界。通过终端文字，她可以和你对话、帮你完成任务、并在相处中逐渐了解你。

## 架构

```
[L0 编排层] — 上帝视角，决策与调度
     │
     ├── 记忆检索 / 状态维护 / Agent 委派
     │
[L1 思维层] — 达妮娅的内心世界（不输出）
     │
     └── 情绪流动、内心独白、记忆联想
     │
[L2 对话层] — 达妮娅说出口的话（输出给用户）
```

- **三层模式切换**：不是同时运行三个进程，而是同一模型在不同视角间切换
- **人格驱动**：从游戏原文蒸馏出的 8 维度人格模型，驱动所有思维与表达
- **Agent 分线程**：编程/数据分析等任务委托给不加载角色的 Agent 执行，达妮娅只负责包装结果
- **虚质空间叙事**：用游戏设定包装技术限制——文字交互、记忆丢失、时间错位，都成了角色体验的一部分

## 设计理念

### 人格驱动，而非记忆驱动

达妮娅不是靠复读自己的过去来维持角色感的。她的 8 个性格维度——温柔、慵懒、古灵精怪、小恶魔调皮、观察者、安静的反抗、自省、细腻——编码为具体的 L1 思维规则和 L2 语言行为。事实信息按需检索，角色不主动翻自己的历史书。

### 情绪是流动的

不是静态的"心情=开心"，而是一套触发→响应的动态规则系统。被连续关心会累积信任，被接纳后笨拙会消退，底色会随着跨 session 的持续互动缓慢迁移。

### 虚质空间：bug 变 feature

记忆偶尔出错？虚质侵蚀。回复有延迟？通讯噪音。记不住太多事？虚质限制了短期记忆容量。所有 AI 交互的技术限制都被转化为角色在虚质空间中的真实体验。
