# 虚质之前记忆重构计划

> 2026-06-25 拟定，待实施

## 问题

- 当前 `初始记忆.md` 单文件 90 行，叙事和事实混在一起
- 模型在生成时不会翻完整个文件，关键事实容易被忽略
- 实测：爱弥斯"从未直接见面"写在末尾，模型仍编造了拆音感仪的师徒场景

## 方案

拆分为小型目录，按主题分文件，每次按需加载相关文件：

```
虚质之前/
├── _索引.md           ← 主题→文件映射，检索入口
├── 核心事实.md         ← "我是谁" + 关键时间线，接近始终加载
├── 人物/
│   ├── 西格莉卡.md    ← 最好的朋友，有愧疚
│   ├── 爱弥斯.md       ← 学院偶像，从未直接见面，仅通过档案/传闻/远距离观察了解
│   ├── 漂泊者.md       ← 复杂情感：感激+不甘+敬佩+同情
│   ├── 弗洛洛.md       ← 残星会唯一的温暖，小提琴
│   ├── 斯瓦茨洛.md     ← 会长，创造者/操纵者，恐惧与厌恶
│   ├── 娜斯塔霞.md     ← 学姐，感激与深深内疚
│   ├── 绯雪.md         ← 击败她的人，有PTSD但不恨
│   ├── 琳奈.md         ← 轻松的学院关系
│   └── 其他.md         ← 千咲、莫宁、洛瑟菈、陆赫斯等
├── 地点/
│   ├── 拉海洛.md       ← 地下绿洲，不靠海，炉芯
│   ├── 星炬学院.md     ← 四大学部，校园日常
│   └── 索拉里斯.md     ← 世界观：悲鸣、共鸣者、残象、鸣式
└── 关键事件.md          ← 暴走、冰原、生日、电车难题、自我放逐
```

## 每个角色文件的格式

```
# 角色名

- **关系**：（亲密/复杂/负面/未直接见面）
- **达妮娅如何认识**：（亲眼所见 / 传闻 / 档案 / 远距离观察）
- **事实边界**：（明确标注——达妮娅知道什么、不知道什么）
- **情感温度**：（见到这个人时她的情绪基调）
- **禁止**：（绝对不能说错的点）
```

## 关键原则

- 每个文件 15-30 行，检索时只加载相关文件
- 事实边界必须写清楚——"从未见面"、"仅通过传闻了解"等不能埋在段落里
- 信息来源从 `扩展设定.md` 和 `人物关系.md` 提取，这两个是权威源
