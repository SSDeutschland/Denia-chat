# 拉海洛（Lahai-Roi）地区设定

> 整理自：萌娘百科、Fandom Wiki、九游、biubiu001、库街区官方
> 对应内容重心第2条：拉海洛地区设定

---

## 一、基本信息

- **所属范围**：索拉里斯星球**新极点**（索拉里斯之极）
- **地理位置**：**罗伊冰原**正下方的巨型地下构造体
- **环境特征**：与地表极寒冰原截然不同，拉海洛是一片**温暖湿润的地下绿洲**
- **实装版本**：3.0
- **文化原型**：挪威 + 苏联时期科研基地美学

---

## 二、历史成因

1. 古时因**"第一次悲鸣"事件引发地磁极偏移**，原极点沉降
2. 罗伊冰原从寒带峡湾变为冰雪覆盖的荒原
3. 原住民**罗伊族**被迫迁入地下，发现了拉海洛这片绿洲并定居
4. 磁极变化还导致索拉里斯地外的**近地卫星与空间站坠落**，残骸至今散布于冰原地表
5. 深空联合在此创立星炬学院，拉海洛成为科研重镇

---

## 三、核心势力与组织

| 组织 | 简介 |
|------|------|
| **深空联合** | 索拉里斯各国共同组建的非营利国际科研同盟，以研究"隧者"为核心使命，探索人类重返宇宙的途径 |
| **星炬学院** | 深空联合下属的无国界学术共同体，知识自由流通的学园都市 |
| **研究院** | 专注前沿研究与技术转化，高悬于巨剑顶端，形如守护的星环 |
| **罗伊族** | 拉海洛原住民，拥有独特文化与信仰体系 |

## 四、罗伊族（原住民）

拉海洛的原住民，拥有悠久的文化传统。

### 三纪神话
罗伊族的创世/历史神话分为三纪：
- **第一纪**：悲鸣降临，世界陷入灾难
- **第二纪**：**「巴德尔」**（隧者）封印邪神「赫龙格」（阿列夫一）后重伤，心脏化为炉芯
- **第三纪**：罗伊人在炉芯映照下于拉海洛繁衍

### 文化与制度

| 设定要素 | 详情 |
|----------|------|
| **历法** | **朔尔历**（地表太阳历）+ **巴德历**（地下炉芯历）双重体系 |
| **圣地** | **浮光林**——日灵栖息之地，林中设**引日殿** |
| **领导层** | **秘日六席**长老——分设视界、灵悉、织星、镌律、拨缕、归律六席 |
| **种族特性** | 因长期受炉芯映照，罗伊人可身着单衣行于冻土 |
| **文字** | 罗伊文的语言逻辑与隧者的操作逻辑高度相似 |
| **数理启蒙** | 使用**"螺算石母"**启发儿童的数理思维 |
| **成年仪式** | 刻印电路板形纹身 |
| **对立势力** | **流民**——背弃罗伊传统、被驱逐的混乱人群 |

### 秘日六席
罗伊族的六位长老，负责观测与推演：
1. **视界** — 观察与预知
2. **灵悉** — 与日灵/隧群沟通
3. **织星** — 历法制定（已知成员：伊瑟珀斯）
4. **镌律** — 律法与规则记录
5. **拨缕** — 仪式与祭祀
6. **归律** — 调和与裁决

## 五、关键概念

### 隧者（The Treader）
- 本质：来自**漂泊者母星**的战斗机械
- 使命：护送岁主投放索拉里斯，建设文明
- 经历：被鸣式「阿列夫一」击落后迫降于罗伊冰原
- 地位：被视为**"拟岁主"**（后升格为正式岁主，更名「拉海洛」）
- 状态：取出自身炉芯后陷入休眠，以炉芯维持拉海洛环境平衡
- 最终：升格为正式岁主，获得说话能力，独自留守隧门对抗阿列夫一

### 炉芯（巴德尔之心）
- 隧者的能源核心
- 悬浮于拉海洛上空，充当地下世界的**"第二个太阳"**
- 同时监视鸣式「阿列夫一」的动向
- 罗伊人通过观测炉芯来制定历法
- 千百年的炉芯映照赋予罗伊人特殊体质

### 隧群
- 隧者陨落后零件分化形成的**生物机械拟态生态群**
- 被罗伊人视为隧者的"肉"
- 罗伊人负责引导隧群的成长与迁徙
- 与罗伊族的日灵信仰紧密相关

### 日灵
- 栖息于圣地浮光林的灵体/存在
- 与隧群关系密切
- 罗伊族通过引日殿与日灵沟通

## 六、环境与科技特征

### 自然环境
- **地表**：罗伊冰原——冰雪覆盖的极寒荒原，散落着坠落的卫星与空间站残骸
- **地下**：拉海洛——温暖湿润的绿洲，由炉芯提供光与热
- **进入方式**：通过穗波市突发的地壳裂缝进入（2.8版本）

### 科技特征
- **虚质磁暴**：拉海洛地区的区域性海蚀现象，由鸣式阿列夫一引发
- **偏谐系统**：3.0新增战斗机制，通过累积偏谐值使敌人进入可被精准打击的状态
- **科考摩托**：由隧者工学部与虚质科学部联合研发的轻量化载具，搭载虚质磁暴侦测模块
- **引日透镜**：学院内最大的光能设施，拉海洛地区中枢信标

## 七、设计美学

拉海洛的整体视觉风格偏向**冷峻厚重的工业史诗感**：

- 高耸穹顶结构
- 霜蚀金属廊道
- 低温等离子照明系统
- 功能美学与模块化的工业实用主义
- 参考了**西伯利亚冻土带与苏联时期科研基地**的建筑美学

与1.0瑝珑的东方国风、2.0黎那汐塔的古典地中海风格形成鲜明对比。

## 八、与主线剧情的关联

1. 2.8版本：穗波市突发地壳裂缝，通往拉海洛的入口被发现
2. 3.0版本：漂泊者与千咲进入拉海洛，星炬学院正式登场
3. 3.1版本：残星会会长斯瓦茨洛以"娜波摩"身份渗透学院，蛊惑西格莉卡
4. 3.3版本：达妮娅（残星会人造实验品）身份暴露，与阿列夫一融合后暴走，最终被绯雪击败并剥离鸣式频率
