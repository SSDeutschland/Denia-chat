## 一、基本概况

| 项目 | 内容 |
|------|------|
| **名称** | 残星会 |
| **英文名** | Fractsidus |
| **日文名** | 残星組織 / フラクトシデス |
| **性质** | 全球性敌对组织，极端科研/宗教团体 |
| **首领** | 斯瓦茨洛 / "卡戎"（Charon） |
| **标志** | 黑洞中嵌着星辰 |

### 名称含义
**FRACTUS（破碎）+ SIDUS（星辰）= FRACTSIDUS**，呼应游戏开场动画中**破碎月球的视觉符号**。

> 核心理念：*"悲鸣不是灾厄，而是人类进化必要的助力。一切都为促成新世界的到来，即使代价是巨大的痛苦。"*

---

## 二、组织目标

| 目标层级 | 内容 |
|----------|------|
| **直接目标** | 不择手段研究鸣式，让鸣式活动以收集数据 |
| **中级目标** | 实现**人类与残像的融合**，推动全人类共鸣者进化 |
| **终极目标** | 引发"真正的悲鸣"，摧毁/重塑索拉里斯世界，迎来崭新世界 |
| **战略核心** | 将索拉里斯人类从**"频率人"转化为"鸣式人"**——使人类能在鸣式影响下保持自我，甚至反过来控制鸣式 |

> 注意：残星会并非纯粹灭世组织——在消灭鸣式的大方向上，甚至与漂泊者有一定共同性。但其手段极端，**不惜以整个地区的毁灭为代价进行实验**。

## 六、技术研究方向

1. **人工诱导共鸣** — 通过非自然手段激发共鸣能力
2. **残像声骸与人类机械耦合** — 将残象肢体与人体融合（面罩信徒、造匠系列）
3. **岁主复制计划** — 试图复制/控制岁主（聚械机偶为已知失败案例）
4. **二次共鸣技术** — 探索人类与残像/鸣式更深层融合
5. **鸣式共鸣容器** — 制造能承载鸣式频率的人造人（达妮娅为成功案例）
6. **鸣式-月相系统研究** — 构建宇宙论层面的行动基础

## 八、与漂泊者的深层关系

1. **"锚点"说**：斯瓦茨洛透露，残星会曾是漂泊者的"锚点"之一——甚至可能是**前文明时期漂泊者自己所创立的组织**
2. **"群星会→残星会"假说**：玩家推测组织原名可能是"群星会"，因漂泊者离去而改名"残星会"（"残"指群星的调律者尚未回归）
3. **伤痕的暗示**：伤痕曾称漂泊者可能以前是残星会的一员
4. **并非全然对立**：会长斯瓦茨洛原话——其目的与漂泊者的道路未必"全然对立"

## 十、核心矛盾

残星会的深层魅力在于其**目标的模糊性**：

- 表面：疯狂科学家式的极端组织，为实验不惜毁灭地区
- 内里：可能在用极端手段做"正确的事"——征服鸣式、让人类进化至能对抗悲鸣
- 立场：与漂泊者（守护现有世界）形成"进化 vs 守护"的根本对立
- 讽刺：在消灭鸣式的大方向上，与漂泊者目标一致，仅手段不同

> 黑海岸 = 守护索拉里斯，以牺牲自我为代价
> 残星会 = 改造索拉里斯，以牺牲他人为代价
