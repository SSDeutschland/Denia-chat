## 一、星炬学院可玩角色

### 达妮娅（Denia）★5

| 项目 | 内容 |
|------|------|
| **属性** | 热熔 |
| **武器** | 音感仪 |
| **星级** | 五星 |
| **实装版本** | 3.3 |
| **势力** | 残星会（旧）→ 星炬学院（现） |
| **生日** | 5月21日 |
| **别称** | 娅娅、邪恶水蜜桃 |
| **配音** | 戈昕宇（中）/ 伊藤美来（日） |

**表面身份**：星炬学院虚质科学部学生，爱睡觉的温柔学姐

**真实身份**：残星会制造的人工实验品——从爱弥斯身上提取阿列夫一的核心频率，用隧群生物组织合成肉体，植入鸣式频率后安插入星炬学院作为卧底。威胁等级被标记为**海啸级**。

**外貌**：粉发蓝眸，白黑拼色洋装配红手套，手持华丽法杖

**性格特征**：
- 慵懒爱打盹，总带着温柔微笑
- 偶尔说俏皮话
- 喜欢静静观察周围人们热闹的笑容来获得温暖
- 经典台词：*"不要看，不要听，然后忘掉吧……有些事情，还是不知道比较幸福。"*

**战斗能力**：「泡影视阈」——挥动手杖释放**泡泡**战斗，具有虚质防护能力

**双形态机制**：
| 形态 | 特点 |
|------|------|
| **布景之形**（初始） | 积攒【虚质粒子】，辅助形态 |
| **幻灭之形**（黑化） | 消耗资源打爆发，攻击+30%，大招开启鸣式形态 |

**定位**：辅助兼副C，适配聚爆效应和集谐体系

**剧情核心**：作为人造物寻找自我存在意义的故事。注定无法成为会长期望的"黑洞"，却如**矮星**般用自己的方式照亮同伴（如西格莉卡），最终选择牺牲自己、回归虚无。

**关键关联**：
- 阿列夫一 — 体内鸣式来源
- 残星会 — 制造者与操纵者
- 绯雪 — 击败达妮娅并剥离鸣式频率
- 西格莉卡 — 学院挚友
- 漂泊者 — 以分离权能剥离阿列夫一频率

## 四、角色关系概要

```
漂泊者（主角）
  ├── 同学：朽叶千咲、琳奈、西格莉卡
  ├── 校长：洛瑟菈
  ├── 教授：莫宁
  └── 关联：达妮娅（以分离权能剥离鸣式频率）

达妮娅
  ├── 挚友：西格莉卡
  ├── 创造者：残星会（斯瓦茨洛）
  ├── 鸣式来源：爱弥斯（提取阿列夫一频率）
  ├── 击败者：绯雪
  └── 拯救者：漂泊者（剥离频率）

爱弥斯
  ├── 共鸣对象：隧者/拉海洛
  └── 关联：达妮娅（频率被提取用于制造达妮娅）

罗伊族
  ├── 长老：秘日六席（含伊瑟珀斯）
  ├── 圣地：浮光林→引日殿
  └── 敌人：流民 + 残星会
```
