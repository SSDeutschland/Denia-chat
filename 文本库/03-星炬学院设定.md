## 二、所属组织：深空联合（Deep Space Union）

### 组织概况
- 由索拉里斯**各国势力共同组建**的非营利国际科研同盟
- 主要使命：研究**"隧者"**、探索重返宇宙的途径
- 最初名为**"极地联合"**，由先行公约提议创建
- 总负责人：**查特**（新联邦出身，前新联邦政府官员）

### 标志含义
- 标志上的**九颗星星**象征天空海覆盖索拉里斯前，人类向天空发射的**九架系外探测器**

### 下设机构
| 机构 | 职能 |
|------|------|
| **星炬学院** | 无国界学术共同体，人才培养与基础探索 |
| **研究院**（研究五所） | 前沿研究与技术转化，高悬于巨剑顶端 |

### 特色装置：「星栈」
- 研究员佩戴的**漂浮型头饰**
- 用于实时接收、监测学员频率信号
- 莫宁的"改良星栈"还可储存自身共鸣能力，实现监控义肢、回复邮件、自动驾驶等功能

## 三、四大学部

### 1. 隧者工学部 —— "最激进的学部"

| 项目 | 内容 |
|------|------|
| **核心使命** | 围绕人形机兵**"隧者"**进行分析研究 |
| **学术风格** | 学院中最"激进"的学部 |
| **下属学科群** | **机体共融学科群** + **战术开发学科群** |
| **研究成果** | 科考摩托、人形机械修造、黑石武器、战术装备 |
| **代表学员** | 朽叶千咲 |
| **代表教授** | 莫宁（兼任生体机械隧群造物研究社顾问） |

### 2. 拉贝尔开发学部 —— "核心学部"（适格者学部）

| 项目 | 内容 |
|------|------|
| **命名** | 以共鸣测定曲线发明者**史考特·拉贝尔**命名 |
| **核心使命** | 发掘培养**"适格者"**以驱动人形机兵隧者 |
| **定位** | **整个星炬学院最初围绕此学部建立** |
| **下属学科群** | **共感训练学科群** + **文明协同学科群** |
| **特殊权限** | 准适格者拥有极高通行权限，可不受限制使用各学部设施 |
| **代表学员** | 漂泊者（主角） |
| **重要职员** | 陆·赫斯（共鸣医疗科负责人/校医） |

> 适格者：从索拉里斯各地选拔，可通过"外源性共鸣"（借助外置机械装置与巨型机体隧者建立意识连接）控制隧者的准驾驶员。

### 3. 光矩能源学部

| 项目 | 内容 |
|------|------|
| **核心使命** | 聚焦**"炉芯"光能**的能源转化技术 |
| **位置** | 学院左区 |
| **下属学科群** | **光能矩阵学科群** + **生态维持学科群** |
| **光能矩阵** | 围绕**"引日透镜"**（拉海洛中枢信标）修建，将炉光折射转化为能源 |
| **生态维持** | 又称"农学部"（被学生视为"第五个学部"），主导"超级宇航食物"计划 |

### 4. 虚质科学部

| 项目 | 内容 |
|------|------|
| **核心使命** | 研究**虚质磁暴**的对策 |
| **标志性建筑** | 学院内最高的信号塔台，顶部为观测拉海洛地块的**天文台** |
| **职责** | 观测预警虚质磁暴、破译生成原理、开发通信与空间锚定技术 |
| **下属学科群** | **空间异变学科群** |
| **代表学员** | 达妮娅（表面身份） |

## 五、重要设施

### 深空文献中心
- 索拉里斯访问权限最高的综合性期刊数据库之一
- 设有人工智能 **I.R.I.S.**
- 先行公约《寰宇人类注疏》编辑部所在地

### 引日透镜
- 学院内最大的光能设施
- 将炉芯的炉光通过折射转化为学院能源
- 拉海洛地区中枢信标

### 信号塔台/天文台
- 虚质科学部标志性建筑，学院最高点
- 观测拉海洛地块与虚质磁暴活动

### 科考摩托
- 隧者工学部 × 虚质科学部联合研发
- 轻量化全地形载具
- 搭载虚质磁暴侦测模块及配套应急防御系统

## 六、学院文化

### 七大不可思议
学院传说中的校园怪谈，目前剧情提及的：
- **I.R.I.S.**：深空文献中心的人工智能，被列为不可思议之一
- 漂泊者被琳奈称为**"第九大不可思议"**（新增）

### 社团
- 飞羽同好社（西格莉卡所属）
- 生体机械隧群造物研究社（莫宁任顾问）
- 多炮塔社（陆·赫斯任顾问）

### 学制
- 2年预科（基础训练+共鸣测定）→ 4年专精（分学部深入学习）
- 知识自由流通，无国界壁垒

---

## 七、设计美学

整体呈现**冷峻、厚重、工业史诗感**：
- 高耸穹顶结构
- 霜蚀金属廊道
- 低温等离子照明系统
- 功能美学与模块化工业实用主义
- 与2.0黎那汐塔的宗教神圣风格形成鲜明对比
