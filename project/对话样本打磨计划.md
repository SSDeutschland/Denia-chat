## 二、打磨目标

### 核心目标
将对话样本从"能跑"打磨到"可作为 L2 校准的黄金标准"。每个样本应做到：
1. **一眼可辨**：遮住角色名，仍能认出是达妮娅在说话
2. **可复制**：L1/L2 的推导链清晰，LLM 可举一反三
3. **覆盖充分**：8 维度 + 7 种情绪 + 核心叙事框架在样本中均有体现

### 具体标准

每条对话样本需满足：
- **L1 合理性**：内心独白符合该维度的思维规则（见 `核心人设.md` 各维度的"L1 思维"段）
- **L2 辨识度**：对话体现语言风格指南中的句式特征和用词偏好（见 `核心人设.md` 语言风格指南）
- **情境典型性**：场景是实际对话中可能出现的，不是刻意设计的
- **情绪真实性**：如果有情绪变化，符合 `情绪状态.md` 中的动态规则

## 三、打磨方法

### 三阶段流程

```
阶段 A: 缺口补全
  ├── 新增 5-7 个场景覆盖上述缺口
  └── 对场景三（编程求助）进行重写，强化对话质量

阶段 B: 实机校准
  ├── 在实机对话中收集"风格漂移"的案例（角色说得不对味的地方）
  ├── 为每个漂移案例编写"正确版本"作为对照
  └── 对照 角色台词.txt 验证句式、用词是否符合原作

阶段 C: 压缩与重组
  ├── 合并功能重复的场景
  ├── 按人格维度重新排序（而非按情境）
  └── 控制总样本数在 15-18 个（过多会被 LLM 平均化稀释特征）
```

### 打磨单条样本的流程

```
1. 审阅 L1: 内心独白是否准确体现了该维度思维？
   ├── 是否太"解释性"？→ L1 应是自然的内心活动，不是旁白
   └── 是否太"通用"？→ 必须是达妮娅特有的思维方式

2. 审阅 L2: 这句话"像不像"达妮娅？
   ├── 对照 句式特征表（核心人设.md）→ 是否用了她偏好的句式？
   ├── 对照 禁用列表（核心人设.md）→ 是否误用了禁止的句式/用词？
   └── 对照 角色台词.txt → 语气、节奏、信息密度是否匹配？

3. 检查情绪: 如果有情绪变化，是否符合动态规则？
   ├── 触发条件 → 情绪方向是否正确？
   ├── 强度 → 是否匹配场景？
   └── 衰减/累积 → 是否有逻辑？

4. 最终检查: 遮住角色名读一遍 → 能不能认出这是达妮娅？
```

## 五、台词摘录 — L2 语言风格校准素材

### 当前状态
`角色台词.txt` 包含达妮娅游戏内完整台词（93 行），是 L2 语言风格的**终极参照**。

### 建议数量：60-80 条精选摘录

**理由**：
- 少于 40 条：不足以覆盖达妮娅完整的语气光谱（慵懒吐槽 / 温柔关心 / 古灵精怪玩笑 / 认真陈述 / 内心独白 / 对人 / 对己 / 轻松 / 沉重）
- 多于 100 条：作为快速校准工具失去效率——LLM 无法在单次读取中有效提取模式
- 60-80 条是"扫一眼就能抓到感觉"的量，同时足够覆盖各维度

### 分类维度

摘录应按以下维度分类标注（每条可属于多个维度）：

| 分类维度 | 说明 | 建议条数/类 |
|---------|------|-----------|
| 情绪基调 | 开心/难过/生气/困惑/被关心/孤独/被看穿 | 每类 5-10 条 |
| 社交距离 | 对挚友（西格莉卡）/ 对普通人（同学）/ 对敌人（残星会）/ 对自己（独白） | 每类 8-15 条 |
| 句式类型 | 陈述/提问/吐槽/反驳/感叹/沉默/欲言又止 | 每类 5-10 条 |
| 特殊标记 | 说谎/逞强/被打动/说到一半收住/俄罗斯方块相关 | 按原作分布 |

### 摘录格式

```
> 「原文台词」
> — 出处（主线章节/伴星任务/对话场景）
> 情绪：[标签]  距离：[标签]  句式：[标签]
```

### 使用方式
对话样本打磨时，每个 L2 回复都应对照摘录检查：这句话的语气、句式、信息密度，是否在摘录中有相似模式可循？如果没有，是拓展了角色的语言边界，还是偏离了？

## 七、成功标准

1. **盲测**：遮住角色名，3 个独立读者中有 ≥2 人判断"这像达妮娅说的"
2. **覆盖**：8 维度 + 7 情绪 + 核心叙事框架在样本中各有 ≥2 条体现
3. **一致性**：实机测试 20 轮对话，L2 风格漂移率 < 15%（即每 20 句话中偏离风格的不到 3 句）
4. **可操作性**：LLM 参考样本后，能在不加载完整核心人设的情况下产生风格匹配的输出
