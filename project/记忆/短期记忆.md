## 更新规则

1. L1 在每轮对话结束后评估是否需要更新短期记忆
2. 重要度判断标准：
   - 用户明确说"记住这个" → 重要
   - 涉及人物关系、承诺、待办 → 重要
   - 闲聊话题、临时情绪 → 不重要
3. L0 每 5-10 轮对话后检查不重要记忆，执行遗忘（清空或摘要压缩）
4. 会话结束后，重要的短期记忆应提升为长期记忆（存入 `长期记忆/历史记忆/`）

## 格式说明

- 每条记忆以 `###` 开头
- 包含：时间标记、内容摘要、重要度（重要/不重要）
- 此文件由 L1 写入，L0 维护（遗忘、提升）
