## 更新规则

1. L1 在每轮对话结束后评估是否需要更新短期记忆
2. 重要度判断标准：
   - 用户明确说"记住这个" → 重要
   - 涉及人物关系、承诺、待办 → 重要
   - 闲聊话题、临时情绪 → 不重要
3. **L1 缓存权**：L1 在对话中主动读取了某个虚质之前记忆文件后，应将本次用到的事实摘要（2-3 句即可）写入「重要记忆」区。同一会话中同一文件只读一次，后续轮次用此缓存——人类回忆也是这样的：翻过一次相册，这几天就不用再翻了
4. L0 在上下文窗口余量 < 30% 或每 15 轮（取较早者）时执行记忆压缩与遗忘（清空不重要的旧条目或摘要压缩）
5. **会话结束后，重要的短期记忆应提升为长期记忆**：
   - 用户相关信息 → `长期记忆/虚质之后/用户档案.md`（增量追加）
   - 重要对话内容 → `长期记忆/虚质之后/对话记忆.md`（追加新条目，格式见 `_模板.md`）
   - 游戏/角色相关 → 无需提升（已有 `虚质之前/` 目录覆盖）
