# 记忆索引

> 关键词 → 记忆文件映射。
> 供记忆检索 Agent 使用，加速定位目标记忆。
> 由 L0 定期维护更新。

---

## 记忆分区

达妮娅的记忆按虚质空间分界线分为两个区间：

| 分区 | 目录 | 内容 | 特征 |
|------|------|------|------|
| **虚质之前** | `长期记忆/虚质之前/` | 游戏剧情记忆（索拉里斯、学院、共鸣等） | 固定不变，核心身份锚点 |
| **虚质之后** | `长期记忆/虚质之后/` | 与用户连接后的记忆（用户信息、对话记录等） | 持续增长，受虚质侵蚀影响 |

检索 Agent 根据查询类型选择优先搜索区间：
- 游戏设定/角色过往 → 优先搜 `虚质之前/`
- 用户信息/当前关系 → 优先搜 `虚质之后/`
- 不确定 → 两者都搜

---

## 索引规则

1. 每个关键词映射到一个或多个记忆文件
2. 关键词从记忆条目的摘要中提取
3. 每次新增或修改长期记忆时，同步更新此索引
4. 检索 Agent 首先读取此文件，确定目标文件后再深入读取
5. 每个文件最多映射 5-8 个关键词，超过时考虑拆分

## 维护说明

- L0 在每次对话结束后评估是否有新记忆需要入索引
- 记忆检索 Agent 检索完毕后可向 L0 提议索引更新
- 此文件保持简洁，每个文件最多映射 5-8 个关键词
- 如关键词过多，考虑拆分记忆文件
- 虚质之后的索引条目随对话自然增长，L0 负责去重和整理
