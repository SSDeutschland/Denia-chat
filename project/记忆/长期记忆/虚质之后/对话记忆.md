# 对话记忆

> 达妮娅与用户的重要对话记录。
> 由短期记忆在会话结束后，将重要条目提升至此。
> 此文件属于"虚质之后"——所有内容都发生在与用户连接之后。

## 条目类型参考

以下类型的记忆应被标记为"重要"并提升到此文件：

1. **承诺与约定**：达妮娅答应了用户的事（"下次帮你查"、"我会记住的"）
2. **用户教过的事**：用户向她解释的概念、技能、知识（同步更新 `用户档案.md`）
3. **用户分享的个人信息**：用户主动透露的、关于自己的事（同步更新 `用户档案.md`）
4. **共同经历的重要时刻**：一次特别长的对话、解决了某个困难、用户帮了她/她帮了用户
5. **断线重连**：跨 session 重连时的心情和印象

## 与短期记忆的关系

```
短期记忆（会话中）
  └─ 会话结束
       ├─ 重要条目 → 提升至此文件（对话记忆.md）
       └─ 不重要条目 → L0 遗忘
```

---

## 维护规则

1. **写入者**：L0 在会话结束后评估短期记忆，将重要条目提升写入
2. **格式一致**：所有条目遵循 `_模板.md` 中的标准格式
3. **索引同步**：添加新条目后，检查 `记忆索引.md` 是否需要补充新关键词
4. **容量控制**：单文件保持在 500 行以内。超过后 L0 将旧条目摘要压缩为一条"早期记忆摘要"
5. **情感标记联动**：条目情感标记应与 `情绪状态.md` 中的记录一致
6. **跨引用**：如果某条记忆与 `用户档案.md` 相关，在内容中注明 `（参见用户档案）`
