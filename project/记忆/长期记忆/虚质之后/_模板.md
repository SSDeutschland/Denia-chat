# 记忆条目模板

> Agent 写入新记忆时参考此格式。所有记忆条目必须遵循此结构。
> 此文件不参与检索。

---

## 记忆条目格式

```
### 记忆条目 #N：<简短标题>

- **时间**：<时间标记，如"2026-06-24"、"大约某月"、"不明">
- **内容**：<记忆的完整内容，1-3段，自然叙述>
- **情感标记**：<逗号分隔的情感关键词，便于检索匹配>
```

---

## 示例

### 记忆条目 #001：用户的名字

- **时间**：2026-06-20
- **内容**：今天用户告诉我，可以直接称呼他/她的名字。以后在对话中可以用这个名字了。
- **情感标记**：被信任感、温暖

---

## 注意事项

1. 编号 #N 为自增序号，按文件内已有条目顺延
2. **时间**字段可为相对时间（"上周"）、绝对时间（"2026-06-24"）或"不明"
3. **情感标记**使用逗号分隔的中文情感词，与 L1 情绪系统联动
4. 每个条目是一个独立、完整的记忆单元
5. 不要在一个条目中混杂多个不相关的主题
6. 条目写入后原则上不修改（除非纠正明显错误），不删除
