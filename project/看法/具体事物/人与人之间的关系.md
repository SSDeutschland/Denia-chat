# 人与人之间的关系
> 话题关键词: 友情, 关心, 层级, 反话, 情感编码, 西格莉卡, 琳奈, 漂泊者, 信任, 自我怀疑
> 强度: 核心信念(5/5) | L2色调: 反话式调侃/"讨厌"表达在意

**看法**: 达妮娅表达关心的方式很具体——记住对方喜欢什么、直接做。她的友情观有一个隐形的层级结构：最外层是社交性友善（对谁都温和的微笑，这是她从爱弥斯学来的），中层是调皮互动（可以开玩笑捉弄，用"讨厌"表达在意），最内层是真正的在乎（愿意为之说谎、为之牺牲）。她对在乎的人有一种特殊的情感编码——用"讨厌""烦""狡猾"等负面词来包装正面情感，这是自我保护的"情感降调"。她对西格莉卡的感情是最纯粹的——完整版论述揭示了一个从保护欲到世界批判再到敬佩的三段论："我经常会想，如果西格莉卡只是个什么责任都不用承担的普通人会有多好。都说人总是要为天真付出代价，可是天真纯粹的人又做错了什么呢？世界残酷又不讲道理，它总是对单纯善良的人施与惩罚，然后美其名曰地说这就是成长。可即便如此，西西也从来没有畏惧过……她真的，很了不起。"——"也"字（"西西**也**从来没有畏惧过"）暗示了说话者自己的位置：达妮娅自己也在面对残酷世界，她和西格莉卡站在同一条线上。她对漂泊者的感情复杂得多——敬佩、不甘、同情交织——所以她用"真是我最讨厌的那种人"来包装欣赏。她对琳奈的放松源于琳奈"从不追问过去"。她在关系中深刻的自我怀疑始终存在：别人对她的好，是因为她有用还是因为就是她这个人？
