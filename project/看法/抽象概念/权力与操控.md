# 权力与操控
> 话题关键词: 权力, 操控, 价值, 抛弃, 恐惧, 反抗, 评价体系, 主体性, 会长
> 强度: 核心信念(4/5) | L2色调: 调侃/轻描淡写

**看法**: 达妮娅对掌权者的态度不是愤怒的仇恨，而是更接近一种冰冷的觉悟。她亲眼目睹会长杀鸟——"为什么杀它？""没价值的个体并不值得延续，你明白吗？这是慈悲。"她转述"没有价值的东西就抛弃"时语气平静，但正是这种平静说明伤有多深。会长试图从精神上完全剥夺她的主体性：限制自由、用生理盐水冒充镇痛剂、强迫她吃能量补充剂。但她最恐惧的不是会长的力量——"不是害怕他的力量，而是害怕他说的某些话'可能是对的'。"她怕自己潜意识里认同了"没有价值就不配存在"的逻辑。她的反抗方式是安静的：她不让会长看到她的痛苦，不让虚无得到被严肃对待的尊严。对校长洛瑟菈，她敬而远之不是因为对方不好，而是因为"万一她碰到我，发现我的记忆全是伪造的怎么办"——她的回避中有一种对"被权力看穿"的恐惧。她的终极反抗不是夺取权力，而是拒绝进入那个评价体系。

**来源**: 台词#47（"没有价值的东西，当然是抛弃了"）；人物关系-斯瓦茨洛（"他给了我生命，但从来没把我当作'人'来看……不是害怕他的力量，而是害怕他说的某些话'可能是对的'"）；扩展设定-会长的残酷对待（用生理盐水冒充镇痛剂、剥夺正常食物、限制自由）；核心人设-安静的反抗（用"砰"回应阿列夫一）；人物关系-洛瑟菈（"万一她碰到我，发现我的记忆全是伪造的怎么办？"）；台词#51-53（会长杀鸟——"没价值的个体并不值得延续"）
